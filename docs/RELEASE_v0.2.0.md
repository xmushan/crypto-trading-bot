# Release Notes - v0.2.0

**发布日期**: 2025-11-28

## 🎯 核心新功能

### 1. 币安合约模拟测试完整支持
- 完全支持币安合约模拟环境，可在 `BINANCE_TEST_MODE=true` 下安全测试策略
- 增强测试模式日志，清晰区分测试交易和实盘交易
- 测试模式下自动连接币安测试网 API

### 2. 本地追踪止损计算器
- 追踪止损计算从 LLM 迁移到本地算法，减少 LLM 幻觉影响
- 支持多币种独立追踪止损策略
- 基于 ATR 的动态止损距离计算，自动适应市场波动
- **可配置 ATR 周期**: 通过 `.env` 文件中的 `TRAILING_STOP_ATR_PERIOD` 配置（默认 7）
  - 支持 3（短期灵敏）、7（平衡推荐）、14（标准周期）
  - 使用长期时间周期（如 4h）的 ATR 数据，更稳定可靠

### 3. 多时间框架指标分析
- 新增 5 个经典时间框架的同步分析：3分钟、5分钟、15分钟、1小时、4小时
- 每个时间框架显示关键指标：EMA20、EMA50、MACD、RSI7、RSI14
- 并行获取数据，优化性能，减少延迟
- 自动优化数据回看天数，避免超过币安 API 1000 根 K 线限制

### 4. 外部 Prompt 管理（开发体验优化）
- 交易策略 Prompt 可从 `prompts/` 目录外部文件加载
- 无需重新编译即可修改交易策略
- 预设多种策略模板（趋势交易、短线交易等）
- 配置项：`TRADER_PROMPT_PATH=prompts/trader_system.txt`

### 5. 账户摘要与持仓汇总
- 新增账户总览功能，显示余额、总权益、可用保证金
- 持仓汇总视图，一目了然当前所有持仓状态
- 优化交易信息展示，提升可读性

## 🔧 优化改进

### API 兼容性
- 适配 DeepSeek 官方 API 的 `json_object` 模式
- 支持从 Markdown 响应中解析纯 JSON

### 风险管理
- 更新交易阈值，提升风险控制能力
- 优化追踪止损参数，提高持仓稳定性

### 技术指标
- 移除冗余的 MACD 信号输出，简化指标数据
- 优化时间周期转换函数
- 增加指标计算测试用例，提高代码质量

### 数据展示
- 优化日内和长期数据表头，提升清晰度
- 更新 Web 仪表板 UI，反映最新功能

## 📚 文档更新

- 更新 CLAUDE.md，记录所有新功能和配置说明
- 新增 `prompts/README.md`，详细说明外部 Prompt 使用方法

## 🔄 从 v0.1.0 升级指南

### 必需操作
1. 更新 `.env` 文件，参考 `.env.example` 添加新配置项：
   - `TRADER_PROMPT_PATH` - 外部 Prompt 文件路径
   - `TRAILING_STOP_ATR_PERIOD` - 追踪止损 ATR 周期（默认 7）

2. 如果使用自定义策略，将策略 Prompt 迁移到 `prompts/` 目录

### 可选操作
- 根据交易风格调整追踪止损 ATR 周期（3/7/14）
- 测试多时间框架指标对策略的影响
- 在测试模式下验证新功能

## ⚠️ 注意事项

- 追踪止损算法已从 LLM 迁移到本地计算，可能导致止损行为略有变化
- 多时间框架指标会增加币安 API 调用次数，请注意 API 限流
- 确保 `.env` 文件包含所有必需的新配置项
